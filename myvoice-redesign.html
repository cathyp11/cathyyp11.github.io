MyVoice Website Redesign

Executive Summary

The previous website crashed without any advanced notice. With a larger user-based poll, the website was overwhelmed with the number of user clicks and pages. In order to address the problems of previous website, we applied a new theme and developed a more engaging website overall. The new information hierarchy and vibrant interactive contents adapted on the current website increased the usage of the website and the number of users.

Overall Changes

  • Added sticky option social media buttons to share page content
  • Simplied footer with important materials
  • Added reader progress bar to show user location
  • Implemented a color palette to induce vividness
